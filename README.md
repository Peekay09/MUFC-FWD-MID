# MUFC-FWD-MID
-This project analyzes Manchester United’s forwards and midfielders using advanced metrics like Expected Goals (xG) and Expected Assists (xA) to evaluate performance, efficiency, and chance creation.

-This repository contains an R-based brief/short tactical analysis of Manchester United's forwards and midfielders.

-Data Source:The raw data (`team-players.csv`) was sourced from **[Understat](https://understat.com/)**,The dataset covers the current English Premier League season from August 2025 up to early April 2026

-You can view the HTML report here: https://peekay09.github.io/MUFC-FWD-MID/
